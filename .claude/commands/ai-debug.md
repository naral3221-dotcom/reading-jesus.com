# AI 팀 에러 디버깅

## 명령어 설명
에러가 발생했을 때 3개 AI가 협력하여 원인을 분석하고 해결책을 제시합니다.

## 입력
$ARGUMENTS (에러 메시지, 스택 트레이스, 관련 코드)

## 실행 프로세스

### 1단계: 에러 분석 (병렬)
동시에 요청:
- **GPT** (`debug_error_gpt`): 스택 트레이스 분석, 근본 원인 추론
- **Gemini** (`solve_error_gemini`): 유사 에러 패턴 매칭, 해결 사례 제안

### 2단계: 해결책 종합 (Claude)
두 AI의 분석을 바탕으로:
1. 가장 유력한 원인 결정
2. 해결 코드 작성
3. 테스트 방법 제안
4. 재발 방지책 정리

### 3단계: ERROR_SOLUTIONS.md 업데이트
해결된 에러를 `agent-rules/ERROR_SOLUTIONS.md`에 자동 기록
