# AI 팀 코드 리뷰

## 명령어 설명
3개의 AI(Claude + Gemini + GPT)가 동시에 코드를 리뷰하고 종합 피드백을 제공합니다.

## 입력
$ARGUMENTS (파일 경로 또는 코드 설명)

## 실행 프로세스

### 병렬 리뷰 요청
동시에 3개 AI에게 리뷰 요청:

1. **Gemini** (`review_code_gemini`): UX/UI 관점, 사용자 경험 개선점
2. **GPT** (`review_code_gpt`): 버그, 보안 취약점, 성능 이슈
3. **Claude**: 아키텍처, 코드 구조, 유지보수성

### 종합 리포트 생성
3개 AI의 피드백을 종합하여:
- 공통으로 지적된 문제점 (높은 우선순위)
- 각 AI별 고유 인사이트
- 개선 액션 아이템 리스트
- 수정된 코드 제안
