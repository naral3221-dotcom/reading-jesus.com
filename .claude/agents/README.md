# 전문 에이전트 시스템

이 폴더에는 프로젝트에 특화된 전문 에이전트가 있습니다.

---

## 에이전트 목록

### 1. **bug-analyzer-fixer** - 버그 분석/수정
**트리거:** 버그, 에러, 안됨, 안돼, 깨짐, error, bug

**전문 분야:**
- 에러 로그 분석
- 근본 원인 파악
- 버그 수정 및 검증

---

### 2. **code-review-expert** - 코드 리뷰
**트리거:** 리뷰해줘, 코드 검토, 개선, 리팩토링

**전문 분야:**
- 코드 품질 검토
- 아키텍처 패턴 준수 확인
- 개선 사항 제안

---

### 3. **supabase-debugger** - Supabase 디버거
**트리거:** Supabase 에러, DB 문제, 저장 안됨, 400/403 에러, RLS

**전문 분야:**
- Supabase HTTP 에러 분석
- RLS 정책 문제 해결
- 데이터 흐름 디버깅

---

### 4. **supabase-migration-expert** - DB 마이그레이션
**트리거:** 테이블 추가, 컬럼 추가, 마이그레이션

**전문 분야:**
- 마이그레이션 파일 생성
- RLS 정책 작성
- 스키마 변경 관리

---

### 5. **nextjs-page-generator** - 페이지 생성
**트리거:** 페이지 만들어, 라우트 추가, 새 화면

**전문 분야:**
- Next.js 14 App Router 페이지 생성
- 클린 아키텍처 패턴 적용
- React Query 훅 연동

---

## 사용 방법

### 자동 호출 (추천)
트리거 키워드가 포함되면 Claude가 자동으로 적합한 에이전트를 호출합니다.

### 직접 지정
특정 에이전트를 명시적으로 호출:
```
"bug-analyzer-fixer 에이전트로 이 에러 분석해줘"
"nextjs-page-generator로 프로필 페이지 만들어줘"
```
