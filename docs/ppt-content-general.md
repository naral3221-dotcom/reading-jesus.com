## 슬라이드 2: 리딩지저스란?

### 365일 성경 통독을 위한 앱

**핵심 기능 3가지**

| 기능 | 설명 |
|------|------|
| 체계적인 통독 계획 | 매일 정해진 분량, 365일 완독 |
| 그룹 묵상 나눔 | 함께 읽고, 함께 나누고 |
| 진행률 추적 | 읽음 체크, 완료 시간 기록 |

📱 스크린샷: 랜딩 페이지

---

## 슬라이드 3: 시작하기

### 앱 접속 방법

**Step 1**: 웹 브라우저에서 앱 URL 접속

**Step 2**: "시작하기" 버튼 클릭

**Step 3**: 회원가입 또는 로그인

💡 **팁**: 크롬/사파리 브라우저 권장

📱 스크린샷: 로그인 화면

---

## 슬라이드 4: 그룹 참여하기

### 초대 코드로 그룹 참여

**Step 1**: 그룹 리더에게 초대 코드 받기
> 예: `ABC123`

**Step 2**: 하단 메뉴 > "그룹" 탭 선택

**Step 3**: "참여" 버튼 클릭

**Step 4**: 초대 코드 입력 후 "참여하기"

✅ 완료! 그룹에 참여되었습니다

📱 스크린샷: 그룹 참여 모달

---

## 슬라이드 6: 성경 읽기

### 오늘의 말씀 읽기

**Step 1**: "성경 읽기" 버튼 클릭

**Step 2**: 성경 본문 화면에서 읽기

**Step 3**: 장/절 이동은 상단 네비게이션 사용

**화면 구성**:
- 상단: 책 이름 / 장 선택
- 본문: 성경 텍스트
- 하단: 이전/다음 장 버튼

📱 스크린샷: 성경 리더 화면

---

## 슬라이드 7: 읽음 체크하기

### 읽기 완료 표시

**Step 1**: 읽기 완료 후 "읽음 체크하기" 버튼 클릭

**Step 2**: 확인 팝업에서 "완료하기" 클릭

**Step 3**: ✅ 체크 완료!

**체크 후 표시**:
- 초록색 체크 아이콘
- 완료 시간 기록 (예: 12월 27일 14:30 완료)

💡 **팁**: 실수로 체크했다면 다시 누르면 해제됩니다

📱 스크린샷: 읽음 체크 전/후 비교

---

## 슬라이드 8: Day 이동하기

### 다른 날짜 말씀 보기

**방법 1: 화살표 버튼**
- ◀ 왼쪽: 이전 Day
- ▶ 오른쪽: 다음 Day

**방법 2: Day 번호 직접 입력**
1. Day 번호 부분 클릭
2. 원하는 숫자 입력
3. Enter 키

**방법 3: 오늘로 이동**
- "오늘로 이동" 링크 클릭

📱 스크린샷: Day 입력 화면

---

## 슬라이드 9: 성경 전체 보기

### 365일 통독 일정 확인

**탭 구성**

| 탭 | 내용 |
|----|------|
| 통독 일정 | 365일 전체 일정 |
| 구약 | 39권 목록 |
| 신약 | 27권 목록 |

**통독 일정 화면**:
- 오늘 Day 하이라이트
- 체크된 날은 초록색
- 각 날짜별 성경책/범위 표시

💡 **팁**: 일정 카드를 **길게 누르면** 바로 체크 가능!

📱 스크린샷: 성경 전체 보기 (통독 일정)

---

## 슬라이드 10: 구약/신약별 보기

### 책별 완료 현황

**구약 탭** (39권)
- 창세기부터 말라기까지
- 각 책별 완료 일수 표시
- 완료된 책은 초록색 테두리

**신약 탭** (27권)
- 마태복음부터 요한계시록까지
- 각 책별 완료 일수 표시

**책 클릭** → 해당 책 성경 리더로 이동

📱 스크린샷: 구약/신약 탭

---

## 슬라이드 11: QT 나눔

### 묵상 함께 나누기

**QT 나눔 열기**:
- 홈 화면 > "QT나눔" 버튼
- 또는 성경 탭 > 말풍선 아이콘

**묵상 작성**:
1. 텍스트 입력창에 묵상 내용 작성
2. "게시" 버튼 클릭

**다른 사람 묵상 보기**:
- 그룹원들의 묵상 목록 확인
- ❤️ 좋아요로 격려
- 💬 댓글로 나눔

📱 스크린샷: QT 나눔 화면

---

## 슬라이드 12: 진행률 확인

### 나의 통독 현황

**홈 화면**:
- 진행률 % 표시
- 진행 바 (시각적)
- X일 / 365일

**성경 탭**:
- 상단: "완료: X/365일"
- 각 책별 완료 현황

**그룹 상세**:
- 그룹 전체 진행률
- 멤버별 현황 (리더만)

📱 스크린샷: 진행률 화면들

---

## 슬라이드 13: 유용한 팁

### 더 편리하게 사용하기

| 팁 | 방법 |
|----|------|
| 빠른 Day 이동 | Day 번호 클릭 → 직접 입력 |
| 빠른 체크 | 통독 일정에서 길게 누르기 |
| 놓친 날 체크 | Day 이동 후 체크 가능 |
| 여러 그룹 참여 | 그룹 탭에서 추가 참여 |
| 알림 설정 | 마이페이지 > 알림 설정 |

💡 **홈 화면에 추가하기**
- 브라우저 메뉴 > "홈 화면에 추가"
- 앱처럼 빠르게 접속!

## 슬라이드 14: 자주 묻는 질문

### FAQ

**Q: 다른 그룹에도 가입할 수 있나요?**
> 네, 여러 그룹에 동시 참여 가능합니다.
> 그룹 탭 > "참여" 버튼으로 추가 가입하세요.

**Q: 읽음 체크를 잘못 했어요.**
> 다시 체크 버튼을 누르면 해제됩니다.

**Q: 지난 Day도 체크할 수 있나요?**
> 네! Day 이동으로 과거/미래 모두 체크 가능합니다.

**Q: 오프라인에서도 사용 가능한가요?**
> 읽기는 가능하지만, 체크/나눔은 인터넷 필요합니다.
