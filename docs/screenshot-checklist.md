## 촬영 팁

### 브라우저 설정
1. Chrome 개발자도구 열기 (F12)
2. 디바이스 툴바 활성화 (Ctrl+Shift+M)
3. iPhone 12 Pro (390x844) 또는 사용자 정의 (375x812)

### 촬영 순서 추천
1. **로그아웃 상태**에서 먼저 촬영
   - 랜딩 페이지
   - 로그인 화면
   - 교회 페이지 (권한 없을 때)

2. **일반 사용자 로그인** 후 촬영
   - 홈, 성경, QT 나눔 등

3. **그룹 리더 계정**으로 촬영
   - 그룹 만들기 모달 (각 단계별)
   - 그룹 관리 화면

4. **교회 관리자 토큰**으로 접속
   - 관리자 페이지 각 탭

### 파일 저장 위치
```
public/images/guide/
├── general/     (일반 사용자용)
├── leader/      (그룹 리더용)
└── church/      (교회 담당자용)
```
