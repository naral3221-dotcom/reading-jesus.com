교회 페이지 URL 구조

교회별 전용 페이지

각 교회는 고유한 코드로 전용 페이지를 가집니다.

URL 용도
/church/ABCD 교회 메인 (묵상 작성)
/church/ABCD/sharing 나눔 목록
/church/ABCD/bible 성경 읽기
/church/ABCD/admin 관리자 페이지

교회 메인 페이지 - 헤더

교회 정보 표시

  • 교회 아이콘 + 교회 이름
  • 교회 주소 (설정된 경우)
  • "교회 등록" 버튼 (로그인 사용자)
  • 또는 "등록 교인" 배지 (이미 등록된 경우)
팁: 교회 등록을 하면 해당 교회의 묵상 나눔에 참여할 수 있습니다.

날짜 선택 카드

통독 일정 네비게이션

  • 좌우 화살표로 날짜 이동
  • 해당 날짜의 읽기 범위 표시
  • 암송 구절 표시 (있는 경우)
  • 보충 주간 표시 (해당 시)
2026년 리딩지저스 공식 일정에 맞춰 매일 읽을 성경 범위가 자동으로 표시됩니다.

권한이 없을 때 화면

비로그인 / 미등록 상태

묵상을 작성하려면 다음 조건 중 하나를 충족해야 합니다:

  • 로그인 + 해당 교회 등록
  • 작성 권한 링크로 접속
  • 익명 작성 허용 교회 (설정에 따라)
QR 코드에 작성 권한을 포함하면 누구나 쉽게 묵상을 작성할 수 있습니다.

짧은 묵상 작성

묵상 작성 폼

항목 설명
이름 작성자 이름 (필수)
묵상 내용 오늘 말씀에서 느낀 점

간단한 형식으로 부담 없이 매일 묵상을 나눌 수 있습니다.

묵상 목록

  • 작성자 이름

  • 작성 시간
  • 좋아요 기능
교회 성도들의 묵상을 한눈에 볼 수 있어 서로 격려가 됩니다.

교인 등록

교회 등록하기

"교회 등록" 버튼을 클릭하면 해당 교회의 정식 교인으로 등록됩니다.

  • 로그인 필요
  • 교회 코드 확인
  • 등록 완료 후 "등록 교인" 배지 표시
등록된 교인은 관리자 페이지에서 진행 현황을 확인할 수 있습니다.
