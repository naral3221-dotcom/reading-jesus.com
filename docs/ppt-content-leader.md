## 슬라이드 3: 그룹 만들기 - 시작

### Step 1: 그룹 생성 화면 열기

**경로**: 그룹 탭 > "만들기" 버튼

**기본 정보 입력**:
- 그룹 이름 (필수)
  - 예: "청년부 성경통독", "가족 말씀나눔"
- 그룹 설명 (선택)
  - 그룹의 목적, 진행 방식 안내
  - 서식 지원 (굵게, 줄바꿈 등)

📱 스크린샷: 그룹 만들기 모달 상단

## 슬라이드 4: 그룹 만들기 - 성경 범위

### Step 2: 읽을 성경 범위 선택

| 옵션 | 설명 | 권장 대상 |
|------|------|-----------|

## 슬라이드 5: 그룹 만들기 - 직접 선택

### 책 직접 선택하기 (고급)

**"직접 선택" 클릭 시**:

```
┌─────────────────────────────┐
│ 구약 (39권)     [전체 선택] │
│ ┌───┬───┬───┬───┬───┐      │
│ │창 │출 │레 │민 │신 │...   │
│ └───┴───┴───┴───┴───┘      │
├─────────────────────────────┤
│ 신약 (27권)     [전체 선택] │
│ ┌───┬───┬───┬───┬───┐      │
│ │마 │막 │눅 │요 │행 │...   │
│ └───┴───┴───┴───┴───┘      │
└─────────────────────────────┘
선택됨: 15권
```

- 책 이름 클릭: 선택/해제
- "전체 선택" 버튼: 구약/신약 전체

📱 스크린샷: 커스텀 책 선택기

## 슬라이드 6: 그룹 만들기 - 읽기 플랜

### Step 3: 읽기 플랜 선택

**리딩지저스 선택 시**: 자동으로 365일

**다른 범위 선택 시**:

| 플랜 | 일수 | 적합한 경우 |
|------|------|-------------|
| 365일 | 365 | 1년 완독 |
| 180일 | 180 | 6개월 완독 |
| 90일 | 90 | 3개월 집중 |
| 직접입력 | 1-1000 | 맞춤 일정 |

**직접입력 예시**:
- 40일 특별 새벽기도
- 100일 신약 통독

📱 스크린샷: 읽기 플랜 선택

---

## 슬라이드 7: 그룹 만들기 - 일정 모드

### Step 4: 일정 진행 방식 선택

**📅 캘린더 모드 (추천)**
- 리딩지저스 2026 공식 일정
- 2026년 1월 12일(월) 시작
- 일요일 휴식 포함
- 모두가 같은 날짜에 같은 본문

**🔢 Day 순차 모드**
- 그룹 시작일부터 Day 1, 2, 3...
- 언제든 시작 가능
- 매일 연속 진행 (휴식일 없음)
- 늦게 시작하는 그룹에 적합

💡 **추천**: 교회 전체 캠페인 → 캘린더 모드
         소그룹 자체 진행 → Day 순차 모드

📱 스크린샷: 일정 모드 선택

## 슬라이드 9: 초대 코드 공유

### 멤버 초대하기

**초대 코드 확인**:
1. 그룹 목록에서 해당 그룹 카드 확인
2. 오른쪽 하단에 초대 코드 표시
3. 코드 클릭 → 자동 복사

**공유 방법**:
- 카카오톡/문자로 공유
- 교회 게시판/공지에 안내
- 모임에서 직접 안내

**예시 메시지**:
```
📖 리딩지저스 그룹에 초대합니다!
그룹명: 청년부 성경통독
초대코드: ABC123

앱 주소: [URL]
```

📱 스크린샷: 초대 코드 복사

## 슬라이드 10: 그룹 관리 - 상세 화면

**관리자 메뉴** (관리자만):
- 그룹 정보 수정
- 멤버 관리
- 그룹 설정

📱 스크린샷: 그룹 상세 화면

## 슬라이드 11: 그룹 관리 - 멤버 관리

### 멤버 현황 확인

**멤버 목록 보기**:
1. 그룹 상세 > 관리자 메뉴
2. 멤버 목록 확인

**각 멤버 정보**:
- 이름 / 프로필
- 진행률 (X일 완료)
- 마지막 활동일
- 역할 (관리자/멤버)

**관리 기능**:
- 역할 변경 (관리자 ↔ 멤버)
- 멤버 제외

📱 스크린샷: 멤버 관리 화면

## 슬라이드 12: 그룹 운영 팁

### 성공적인 그룹 운영 비결

**1. 정기 격려**
- QT 나눔에 댓글 달기
- 매일 또는 주 2-3회
- 진심 어린 응원

**2. 진행 현황 공유**
- 주간 진행률 공유
- 밀린 분들 개별 연락
- 격려 메시지

**3. 마일스톤 축하**
- 50일, 100일, 200일...
- 완독 시 축하
- 온/오프라인 나눔 모임

**4. 어려움 해결**
- 밀린 분들: "과거 Day도 체크 가능해요"
- 바쁜 분들: "10분만 투자해보세요"

## 슬라이드 16: 자주 묻는 질문

### FAQ (관리자용)

**Q: 그룹 설정을 나중에 변경할 수 있나요?**
> 그룹 상세 > 관리자 메뉴에서 일부 수정 가능합니다.

**Q: 초대 코드를 바꿀 수 있나요?**
> 현재는 변경 불가능합니다. 기존 코드를 계속 사용해주세요.

**Q: 멤버가 탈퇴하면 어떻게 되나요?**
> 멤버의 진행 기록은 유지되며, 멤버 목록에서만 제외됩니다.

**Q: 여러 그룹을 만들 수 있나요?**
> 네, 제한 없이 여러 그룹을 만들 수 있습니다.
