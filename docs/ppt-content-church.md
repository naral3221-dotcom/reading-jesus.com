## 슬라이드 3: 교회 페이지 URL 구조

### 교회별 전용 페이지

**URL 형식**: `/church/{교회코드}`

| URL | 용도 |
|-----|------|
| `/church/ABCD` | 교회 메인 (묵상 작성) |
| `/church/ABCD/sharing` | 나눔 목록 (짧은 묵상 + QT) |
| `/church/ABCD/bible` | 성경 읽기 |
| `/church/ABCD/admin` | 관리자 페이지 |

**교회 코드**: 대문자 4~6자리 (예: ABCD, GRACE, HOPE123)

## 슬라이드 7: 묵상 작성 폼

### 짧은 묵상 나눔 작성

**입력 필드**:
1. **이름**: 작성자 이름 (등록 교인은 자동 입력)
2. **묵상 내용**: 리치 에디터 (서식 지원)
3. **익명 옵션**: 익명 작성 허용 시 체크박스

**등록 교인 표시**:
```
[이름 입력]  등록 교인
└── 자동 입력, 수정 불가
```

**작성 버튼**: "나눔 등록"

## 슬라이드 10: QT 나눔 작성

### 구조화된 QT 작성

**QT 작성 다이얼로그**:

1. **날짜 선택**: 드롭다운으로 QT 날짜 선택
2. **QT 컨텐츠 보기**:
   - 오늘의 말씀 (접기/펴기)
   - 묵상 길잡이 (접기/펴기)
   - 묵상 질문 (접기/펴기)

3. **나의 묵상 작성**:
   | 항목 | 설명 |
   |------|------|
   | 1️⃣ 내 말로 한 문장 | 말씀 요약 |
   | ❓ 묵상 질문 답변 | 질문에 대한 나의 생각 |
   | 🙏 감사와 적용 | 적용점 |
   | 🙌 나의 기도 | 기도문 |
   | 📋 하루 점검 | 말씀이 삶에 적용된 점 |

📱 스크린샷: QT 작성 다이얼로그

## 슬라이드 12: 교인 등록 기능

### 교회 등록하기

**등록 과정**:
1. 로그인 상태에서 교회 페이지 접속
2. 헤더의 "교회 등록" 버튼 클릭
3. 확인 팝업에서 정보 확인
4. "등록하기" 버튼 클릭

**등록 후 혜택**:
- ✅ QR 코드 없이도 묵상 작성 가능
- ✅ 프로필과 묵상 연결
- ✅ "등록 교인" 배지 표시

## 슬라이드 18: 관리자 - 소그룹 관리 ⚠️ (개발중)

### 교회 소그룹 관리

> ⚠️ **현재 개발 진행 중인 기능입니다**

**소그룹 기능의 목적**:
| 목적 | 설명 |
|------|------|
| **부서별 운영** | 청년부, 장년부, 구역별 등 부서 단위 통독 |
| **특별 QT/묵상** | 소그룹별 특별 QT 혹은 묵상 그룹 진행 가능 |
| **동기 부여** | 함께 읽는 멤버들과 격려 |
| **맞춤 일정** | 그룹마다 다른 시작일/플랜 설정 가능 |

**활용 예시**:
- 청년부 365일 통독반
- 새벽기도 90일 집중 통독
- 구역별 소그룹 말씀 나눔
- 신입 교인 신약 180일 통독

**그룹 탭 기능** (예정):
- 그룹 목록 조회
- 그룹 만들기 (+ 버튼)
- 각 그룹 진행률 확인
- 초대 코드 복사

**그룹 카드** (예정):
```
┌─────────────────────────────────────┐
│ 청년부 통독반          [공식]       │
│ 구역 통독 모임입니다                │
│                                     │
│ 👥 12명 · 📅 365일 · 🎯 진행률 45% │
│ [████████░░░░░░░░░░░░]              │
│                        [📋] [⚙️] [>]│
└─────────────────────────────────────┘
```

## 슬라이드 25: FAQ

### 자주 묻는 질문

**Q: 교회 코드는 어떻게 만드나요?**
> 시스템 관리자에게 요청하여 생성합니다.

**Q: QR 코드를 변경하고 싶어요.**
> 관리자 설정에서 "작성 권한 토큰 재발급"을 하면 새 QR 코드가 필요합니다.

**Q: 교인 등록은 어떻게 하나요?**
> 1) 교인이 리딩지저스 앱에 회원가입
> 2) 교회 페이지에서 "교회 등록" 버튼 클릭

**Q: 부적절한 글을 삭제하고 싶어요.**
> 관리자 페이지 > 묵상글 탭에서 삭제 가능합니다.

**Q: 여러 담당자가 관리할 수 있나요?**
> 관리자 링크를 공유하면 여러 명이 접근 가능합니다.
