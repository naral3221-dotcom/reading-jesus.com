## 📱 QR 코드 찍는 방법 (1단계)

### 스마트폰 카메라로 QR 코드 찍기

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   1. 스마트폰의 [카메라] 앱을 실행합니다                         │
│      (사진 찍을 때 쓰는 앱이에요)                                │
│                                                                 │
│   2. 주보에 있는 네모난 QR 코드에 카메라를 갖다 대세요           │
│      (카메라가 QR 코드를 인식할 때까지 가만히 기다리세요)        │
│                                                                 │
│   3. 화면 위에 노란색 글씨가 나타나면                            │
│      그 글씨를 손가락으로 톡! 누르세요                           │
│                                                                 │
│   4. 자동으로 인터넷 화면이 열리면서                             │
│      우리 교회 페이지가 나타납니다                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**💡 잘 안 되시나요?**
- QR 코드가 너무 멀면 인식이 안 돼요. **가까이** 대세요
- 흔들리면 인식이 안 돼요. **가만히** 잡으세요
- 어두우면 인식이 안 돼요. **밝은 곳**에서 찍으세요

## 🏠 교회 화면 확인하기 (2단계)

QR 코드를 찍으면 **교회 페이지**가 바로 나타납니다!

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   화면에 이렇게 보여요:                                         │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │ 🏛️ OO교회                              [등록 교인]      │   │
│   │    서울시 OO구 OO동...                                  │   │
│   └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │           TODAY'S READING                               │   │
│   │                                                         │   │
│   │      <        📅 1월 20일 (화)        >                 │   │
│   │                  창 29-32                               │   │
│   └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│   → 오늘 읽을 성경 말씀이 표시되어 있어요!                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**💡 화면 위쪽에서 오늘 읽을 성경 범위를 확인하세요**
- 예: "창 29-32" = 창세기 29장부터 32장

## ✍️ 짧은 묵상 나눔 작성하기 (3~5단계)

화면을 조금만 내리면 **"짧은 묵상 나눔 작성"** 칸이 보입니다.

### 3단계: 이름 확인하기

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   📖 짧은 묵상 나눔 작성                                        │
│                                                                 │
│   이름                                                          │
│   ┌─────────────────────────────────────┐                       │
│   │  이재훈                             │    [등록 교인]        │
│   └─────────────────────────────────────┘                       │
│                                                                 │
│   → 이미 이름이 들어가 있으면 그대로 두세요                      │
│   → 이름이 비어있으면 이름 칸을 톡! 눌러서 이름을 쓰세요        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**💡 이름 칸에 이미 이름이 있나요?**
- 있으면 → 그대로 두세요! (이전에 쓴 이름이 저장된 거예요)
- 없으면 → 칸을 톡! 눌러서 이름을 쓰세요

---

### 4단계: 묵상 내용 쓰기 (가장 중요!)

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   묵상 내용                                                     │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │ B  I  U  S  &  |  H₂  ≡  ≡  "  |  ↩  ↪               │   │
│   ├─────────────────────────────────────────────────────────┤   │
│   │                                                         │   │
│   │  (여기에 오늘 말씀 읽고 느낀 점을 쓰세요)               │   │
│   │                                                         │   │
│   │                                                         │   │
│   │                                                         │   │
│   └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│   1. 이 큰 칸(하얀 부분)을 손가락으로 톡! 누르세요              │
│   2. 키보드가 올라옵니다                                        │
│   3. 오늘 말씀에서 느낀 점을 적어주세요                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**📝 뭘 써야 할지 모르겠다고요? 이렇게 써보세요!**

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   ✨ 예시 1 (아주 짧게)                                         │
│   "오늘 말씀 감사합니다"                                        │
│                                                                 │
│   ✨ 예시 2 (조금 더 쓰기)                                       │
│   "오늘 말씀을 읽으며 하나님이 저를 사랑하신다는 것을            │
│    다시 한번 느꼈습니다. 감사합니다."                            │
│                                                                 │
│   ✨ 예시 3 (기도로 쓰기)                                        │
│   "주님, 오늘도 말씀으로 힘 주셔서 감사합니다.                   │
│    이 말씀대로 살게 해주세요. 아멘"                              │
│                                                                 │
│   💡 한 줄만 써도 됩니다! 부담 갖지 마세요                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**✅ 익명으로 쓰고 싶으시면?**
```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   ○ 익명으로 작성                                               │
│                                                                 │
│   → 이 동그라미(○)를 톡! 누르면 체크(●)가 됩니다                │
```

## 📖 더 자세한 QT를 쓰고 싶다면?

화면 맨 아래에 **[QT 작성]** 버튼이 있어요.
더 깊은 묵상을 남기고 싶을 때 사용하세요!

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│         더 자세한 묵상을 남기고 싶다면?                          │
│                                                                 │
│   ┌─────────────────┐      ┌─────────────────┐                  │
│   │                 │      │                 │                  │
│   │   📝 QT 작성    │      │   📖 QT 목록    │                  │
│   │   오늘의 묵상   │      │   전체 일정     │                  │
│   │                 │      │                 │                  │
│   └─────────────────┘      └─────────────────┘                  │
│         ↑                                                       │
│    이 버튼을 눌러요                                              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### QT 작성 순서 (6단계)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  ① 성경 구절        ② 한 문장 요약        ③ 묵상                          │
│  ┌──────────────┐   ┌──────────────┐   ┌──────────────┐                    │
│  │ 오늘 읽은    │   │ 가장 마음에  │   │ 말씀을 통해  │                    │
│  │ 성경 범위    │ → │ 남는 말씀    │ → │ 깨달은 점    │                    │
│  │ (자동 입력)  │   │ 한 줄로 적기 │   │ 적용할 점    │                    │
│  └──────────────┘   └──────────────┘   └──────────────┘                    │
│                                                                             │
│                           ↓                                                 │
│                                                                             │
│  ⑥ 제출 버튼        ⑤ 기도              ④ 감사                            │
│  ┌──────────────┐   ┌──────────────┐   ┌──────────────┐                    │
│  │ [QT 등록]    │   │ 오늘의 기도  │   │ 오늘 감사한  │                    │
│  │ 버튼을 꾹    │ ← │ 적기         │ ← │ 것 적기      │                    │
│  │ 누르기       │   │ (선택)       │   │ (선택)       │                    │
│  └──────────────┘   └──────────────┘   └──────────────┘                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

**💡 모든 칸을 채우지 않아도 돼요!**
- **필수**: 한 문장 요약, 묵상
- **선택**: 감사, 기도, 하루 점검

**📝 각 칸에 뭘 쓰면 될까요?**

| 칸 | 예시 |
|---|---|
| **한 문장 요약** | "하나님이 야곱과 함께하셨다" |
| **묵상** | "나도 어려울 때 하나님이 함께 하심을 믿겠습니다" |
| **감사** | "오늘도 건강하게 예배드릴 수 있어서 감사합니다" |
| **기도** | "주님, 오늘 하루도 함께 해주세요. 아멘" |

## ❓ 이런 경우에는요?

### "글씨가 너무 작아요"

```
손가락 두 개를 화면에 대고 벌리세요 (확대됩니다)
반대로 오므리면 축소됩니다
```

### "실수로 잘못 썼어요"

```
괜찮아요!
내용을 지우려면 키보드의 [←] 버튼(지우기)을 계속 누르세요
다 지우고 다시 쓰시면 됩니다
```

### "버튼이 안 눌려요"

```
손가락으로 버튼 가운데를 꾹~ 1초 정도 누르세요
살짝 터치하면 안 될 때가 있어요
```

### "화면이 갑자기 꺼졌어요"

```
스마트폰 옆면의 버튼을 눌러 화면을 다시 켜세요
그리고 처음부터(QR 코드 찍기) 다시 하시면 됩니다
```

### "인터넷이 안 돼요"

```
Wi-Fi나 데이터가 꺼져 있을 수 있어요
- 교회 Wi-Fi에 연결하시거나
- 화면 위에서 아래로 쓸어내려 데이터 버튼을 켜세요
```

### "나눔 등록 버튼이 안 보여요"

```
화면을 조금 아래로 내려보세요
주황색 [나눔 등록] 버튼이 보일 거예요
```
